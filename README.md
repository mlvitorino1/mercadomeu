# MercadoMeu

## Objetivo do Produto

MercadoMeu é um aplicativo mobile inteligente que transforma cupons fiscais em dados organizados e insights financeiros úteis.

O usuário tira uma foto do cupom fiscal e o sistema:

- Extrai os dados automaticamente via OCR + IA
- Organiza compras e produtos
- Mostra histórico de gastos
- Exibe dashboards inteligentes
- Ajuda o usuário a economizar

---

## Stack Atual

### Frontend
- Lovable

### Backend (Inicial)
- Lovable Cloud Functions / Backend interno

### Banco de Dados
- Lovable Cloud DB

### OCR / IA
- Tesseract OCR
- NLP para estruturação
- IA para normalização de produtos
